### Who can participate

Anyone!

### How to participate

1. Fork this repo.
2. Find a video which needs translation
3. Find the English subtile (such as QiGN5pey8TM_EN.srt)
4. Translate the subtitle in to Chinese, add the srt file (such as QiGN5pey8TM_ZH.srt) to the same folder.
5. Commit and PR.
